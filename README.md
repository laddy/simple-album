# simple-album

シンプルな画像管理ツール

## 動作環境

- PHP5.4 以上
- MySQL5.5 以上
- ApacheやらNginxが動作する環境でDocrootに突っ込んで下さい
- Codeigniterを使っているのでセットアップ方法などは適度調べて下さい

## Setup

- MySQLにpictureというDBを作成してsqlを実行
- application/config/database.php にDB接続の内容が書いてあるので適宜修正
